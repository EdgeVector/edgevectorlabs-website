# edgevectorlabs-website

Corporate website for **Edge Vector Labs LLC** — edgevectorlabs.com.

Labs is the commercial arm of the Edge Vector ecosystem (wholly owned by the
Edge Vector Foundation) and the operator of Exemem. This is a small Vite +
React site in the same design family as edgevector.org
(EdgeVector/edgevector-website), deployed on Vercel.

```bash
npm install
npm run dev      # local dev
npm run build    # production build → dist/
```
